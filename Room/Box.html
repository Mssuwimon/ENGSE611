CSS Box Model

1) box-sizing: content-box (ค่าปกติ)

ความกว้างที่กำหนด = 200px (ไม่รวม padding + border)

2) box-sizing: border-box

ความกว้างที่กำหนด = 200px (รวม padding + border อยู่ใน 200px)

3) เงาธรรมดา (box-shadow)

box-shadow: 10px 10px 5px rgb(0,0,0,0.4);

4) เงาด้านใน (Inset)

box-shadow: inset 0 0 15px rgb(0,0,0,0.4);

5) เงาฟุ้งนุ่ม (box-shadow)

box-shadow: 0 0 20px rgb(0,0,0,0.3);

6) เงาหลายชั้น (box-shadow)

box-shadow: multi-layer 3px 3px 5px rgb(0,0,0,0.2), 6px 6px 10px rgb(0,0,0,0.2);

7) เงาทิศทางลงล่าง (box-shadow)

box-shadow: 0 12px 8px rgb(0,0,0,0.);
